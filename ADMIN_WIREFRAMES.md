## Desktop-First Design Principles

1. **No Mobile Optimization:** Admin uses desktop/laptop only
2. **Multi-Panel Layouts:** Master-detail, split views
3. **Dense Tables:** More rows per page (50 instead of 10)
4. **Keyboard Shortcuts:** Ctrl+K for search, Tab navigation
5. **Hover States:** Rich tooltips, inline editing
6. **Fixed Headers:** Sticky table headers when scrolling
7. **Side Panels:** Details open in side panel (not bottom sheet)
